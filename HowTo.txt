HOWTO

1.
 git clone https://github.com/lukeb1961/PSHardwareInfo.git
 CD .\PSHardwareInfo

2.
# synch down the Hardware.Info repository
 git submodule init
 git submodule update

3.
cd Hardware.Info and compile Hardware.Info.sln using dotnet build Hardware.Info.sln (or VS2022)
this generates the DLL that does all the work

4.
compile PSHardwareInfo.sln using dotnet build PSHardwareInfo.sln (or VS2022)
this generates the DLL that is the PowerShell cmdlet.

5.
using a Powershell session 
create the module manifest and copy the dlls.
.\New-PSmodule.ps1

6.
then deploy the Powershell module, using 
.\Install-PSmodule.ps1

7.
at this point, you should be able to
PS> Import-Module PSHardwareInfo
PS> Get-HardwareInfo
